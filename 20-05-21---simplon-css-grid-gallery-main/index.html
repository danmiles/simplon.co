Hello, welcome.
Here I would like to demonstrate the wide range of capabilities of the CSS Grid responsive website technology.

This technology appeared in 2017 and is the newest and most functional technology for making a website adaptive to different devices (phone, tablet, personal computer monitor). But so far this technology has not found a wide application in commercial projects.

An example would be a simple photo gallery in which the images will change size, the structural grid will also change. This is a simple example, the capabilities of CSS Grid are much more.

All you need to do is shrink your browser screen or use the "toggle device toolbar".
